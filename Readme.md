# Expense Tracker 

The web version of a CLI Expense Tracker is like using the same tool but in a browser. You can add, view, and delete expenses using simple forms and buttons instead of typing commands. All your data is saved, and you can see summaries or charts to track your spending, making it easier and more visual than the command-line version.
